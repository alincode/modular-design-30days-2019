# 模組的 README 設計

README 是模組的門面，能不能吸引要別人使用你的模組，就看這一頁了。

## slogan

通常第一行都是口號 (slogan)，用一句最吸睛的話，來介紹你的模組。

## badges

口號後面通常接著徽章 (badges)，讓使用者可以一目瞭然知道模組目前的狀態。
