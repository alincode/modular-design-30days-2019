# 怎麼挑選外部模組

除了開發自己的模組之外，我們也經常使用別人的模組，但選擇這麼多，要怎麼選呢？

我會以外顯資訊跟內隱資訊來看來評估。

### 外顯資訊

- 下載人數
- 文件完整度
- 測試涵蓋率
- Dependents 數量
- 最新一次的更新時間
- release 頻率
- 星星數 (option)

---

### 內隱資訊

- issue 和 PR 裡的 reply 熱度
- 有沒有重大 bug
- 作者的創作動機跟野心
- 是否有背後資源
- Contributing 文件: <https://github.com/webdriverio/webdriverio/blob/master/CONTRIBUTING.md>
- 作者在不在意 code style
  - 靜態程序分析工具
    - lint 檢查：有沒有 eslintrc.js 檔案
  - 統一編碼風格
    - 有沒有 .editorconfig 檔案
